NAME

chan - terminal-based (ncurses) read/write Hacker News client  

SYNOPSIS

chan [OPTION]... [CONFIG-FILE]...
chan -h|--help
chan -v|--version
 

DESCRIPTION

chan is a client for Hacker News that attempts to implement all features accessible from the web interface. It is designed to be used on text terminals and controlled entirely via key bindings, many of which were inspired by existing programs such as vi(1), less(1), mutt(1), and newsbeuter(1). It also aims to be as easily configurable as possible such that its users can change any aspect of its execution as desired.

The source code is written in C (hence C HAcker News) and licensed under the GNU GPL 3.0.  

OPTIONS

-u, --username
-p, --password
Specify user credentials to log in with.
-h, --help
Output a help message.
-v, --version
Output the currently installed version of chan.
 

CONFIGURATION

The syntax of a configuration file is one option per line, where the format of each line is as follows: 

name=value

The following configuration options (which take the place of name) are available:

username
password
Specify user credentials to log in with.
 

FILES

$HOME/.chanrc
%USERPROFILE%/_chanrc
The default configuration file, for UNIX-like and Windows systems respectively.
If no configuration files are specified on the command line, chan will read from this file by default, if it is present.
